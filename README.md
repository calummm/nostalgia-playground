# Monty Python Larch Site

Welcome to the Monty Python Larch Site! This project is dedicated to celebrating the iconic "Larch" tree from Monty Python's comedic sketches. 

## Project Overview

This website is designed to reflect the nostalgic aesthetic of early 2000s web design, complete with vibrant colors, animated GIFs, and a playful layout. It serves as a tribute to the humor and creativity of Monty Python, particularly focusing on the memorable references to trees, especially the Larch.

## File Structure

The project consists of the following files:

- **src/index.html**: The main homepage of the site, featuring links to the Larch page and including references to CSS and JavaScript files.
- **src/pages/larch.html**: A dedicated page that provides detailed information about the Larch tree, styled in a way that captures the essence of early web design.
- **src/styles/style.css**: The CSS file that contains styles for the webpage, emphasizing bright colors and large fonts typical of the early 2000s.
- **src/scripts/main.js**: A JavaScript file that adds simple animations and interactions to the webpage, reminiscent of the web practices from that era.

## How to View the Site

1. Clone the repository to your local machine.
2. Open the `src/index.html` file in your web browser.
3. Navigate through the site to learn more about the Larch tree and enjoy the Monty Python humor!
